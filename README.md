# 🍳 מטבח·לאב (FoodHelper)

אתר מתכונים, קורסי אפייה ומדריך מסעדות — בעברית, ללא AI וללא API.

## ✨ מה יש באתר

- **📖 150 מתכונים** אמיתיים ומפורטים ב-10 קטגוריות (קינוחים, עוגות, עוגיות, לחמים, מאפים מלוחים, מנות עיקריות, פסטה, סלטים, מרקים וארוחת בוקר)
- **⭐ מועדפים** — סימון מתכונים אהובים (נשמר בדפדפן)
- **🖨️ הדפסת מתכון** בפורמט נקי
- **🎓 קורסי אפייה מותאמים אישית** — בוחרים רמה, עונים על 7 שאלות ומקבלים תוכנית אימון של 4 שבועות עם מעקב התקדמות
- **🍴 מדריך מסעדות** — 53 מסעדות ב-9 ערים בגוש דן וירושלים, עם ניווט ב-Waze ובגוגל מפות
- **🌙 מצב כהה** אוטומטי

## 🛠️ טכנולוגיה

HTML / CSS / JavaScript בלבד — אתר סטטי, בלי שרת, בלי תלויות ובלי מפתחות API.

## 🚀 הרצה מקומית

פשוט פותחים את `index.html` בדפדפן, או:

```bash
python -m http.server 8110
# ואז גולשים ל-http://localhost:8110
```

## 📁 מבנה הקבצים

| קובץ | תפקיד |
|------|-------|
| `index.html` | הדף הראשי (3 עמודים: מתכונים, קורסים, מסעדות) |
| `recipes.js` | מאגר 150 המתכונים |
| `data.js` | קטגוריות, רמות, שאלות החידון ותוכניות הקורס |
| `restaurants.js` | מדריך המסעדות לפי ערים |
| `app.js` | הלוגיקה של האתר |
| `style.css` | העיצוב (כולל מצב כהה והדפסה) |
